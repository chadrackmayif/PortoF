Placez ici vos fichiers PDF de certificats.

Noms recommandés (correspondent aux liens dans la page):
- CCNA.pdf
- RHCE.pdf
- AWS_Solutions_Architect.pdf
- CISSP.pdf

Exemple :
certificates/CCNA.pdf
certificates/RHCE.pdf

Si vous utilisez des noms différents, mettez à jour les attributs `href` dans `index.html` pour pointer vers les bons fichiers.
